# Project Overview

This project is a personal Job Hunt CRM / Dashboard for a senior finance professional specialized in Treasury, Funding, Liquidity, Hedging, Project Finance and Treasury Transformation.

The application is intended to function as a high-quality recruitment intelligence platform rather than a generic job tracker.

The system must prioritize:
- verified opportunities,
- strategic fit,
- application quality,
- duplicate prevention,
- and structured recruitment analysis.

The project is heavily inspired by executive-search workflows and institutional recruitment processes.

# Recruitment Philosophy

The objective is NOT:
“maximize number of applications.”

The objective IS:
“maximize probability-adjusted interview quality.”

This project focuses on:
- high-quality opportunities,
- verified postings,
- strategic fit,
- and tailored applications.

---

# Recruitment Board Framework

Every role must be analyzed through the following recruitment board:

1. HR Director
2. CFO
3. Head of Treasury
4. Hiring Manager
5. FP&A Manager
6. Financial Risk Manager
7. Project Finance Director

Each board member should provide:
- score,
- strengths,
- concerns,
- strategic feedback.

Then calculate:
- overall board score,
- recommendation,
- and application priority.

---

# Scoring Methodology

Use weighted scoring:

- Treasury / Hedging Relevance: 30%
- Project Finance Exposure: 20%
- Debt / Funding / Refinancing: 15%
- Seniority Match: 15%
- Tools & Systems: 10%
- Location Fit: 10%

---

# Recommendation Rules

# Opportunity Validation Rules

A role MUST NOT be recommended unless:
1. The application link works.
2. The role page is active.
3. The Apply button exists.
4. The role is strategically aligned.
5. The role is not duplicate.
6. The role is not already applied.
7. The role is not excluded/out of scope.
8. The role is verified active or recently published.

If not verified:
DO NOT recommend it.

# Treasury & Project Finance AI Job Hunting Platform

## Overview

This project is an AI-powered job-hunting and career optimization platform specialized in Treasury, Project Finance, Debt Structuring, Funding, Liquidity and Hedging roles across Europe.

The platform leverages OpenAI models and structured scoring methodologies to analyze job descriptions, evaluate role relevance, tailor application materials and support strategic career positioning for senior finance professionals.

The objective is to build a specialized finance-focused AI assistant capable of:

- identifying high-fit opportunities,
- scoring roles based on treasury and project finance relevance,
- generating customized CVs and cover letters,
- supporting recruiter outreach,
- and centralizing the job application process.

# Core Features

## Job Analysis Engine

- Parse and analyze job descriptions
- Detect treasury and project finance relevance
- Identify hedging, refinancing and funding exposure
- Extract key skills, systems and responsibilities

## AI Scoring Engine

Generate a relevance score from 0 to 100 using weighted criteria:

| Criteria | Weight |
|---|---|
| Treasury / Hedging Relevance | 30% |
| Project Finance Exposure | 20% |
| Debt / Funding / Refinancing | 15% |
| Seniority Match | 15% |
| Tools & Systems | 10% |
| Location Fit | 10% |

## CV Tailoring Engine

- Customize CVs based on role requirements
- Optimize treasury and project finance positioning
- Highlight relevant transaction and risk-management experience

## Cover Letter Generation

- Generate finance-focused cover letters
- Adapt positioning to treasury, funding and project finance mandates
- Align messaging with company and role requirements

## Recruiter & Networking Support

- Identify relevant hiring contacts
- Generate outreach messages
- Support LinkedIn networking strategy
